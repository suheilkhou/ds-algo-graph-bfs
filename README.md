# DS&A — Graph Modeling and BFS-Based Algorithms

## About the course

This repository is part of my coursework for **Data Structures and Algorithms**, a university course on algorithm analysis and design. Topics include asymptotic analysis, sorting, balanced search trees, hashing, graphs and shortest-path algorithms, dynamic programming, and amortized analysis — with both written theoretical exercises and a final programming project.

## About this assignment

This exercise is about taking word problems and modeling them as graph problems, then solving them with the right traversal. The flagship question models a campus navigation scenario (bus stops as vertices, one-way paths as directed edges) and solves it by adding an auxiliary source vertex connected to every starting faculty, then running a single BFS to extract the answer. Each solution includes the reduction, the algorithm, a correctness argument, and a runtime analysis.
